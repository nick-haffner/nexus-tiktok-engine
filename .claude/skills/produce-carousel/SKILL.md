# Produce Carousel

Read `tiktok_module/procedures/produce-carousel/overview.md` for the tool stack and acceptance criteria, then read `tiktok_module/procedures/produce-carousel/brand-rules.md` and `tiktok_module/procedures/produce-carousel/frameworks.md` for voice and format context.

Execute the phases in order:

1. Read `tiktok_module/procedures/produce-carousel/phase-0-ideation.md` and execute it. Present concept options to the manager for approval before proceeding.
2. Read `tiktok_module/procedures/produce-carousel/phase-1-copy.md` and execute it. Present copy for approval before proceeding.
3. Read `tiktok_module/procedures/produce-carousel/phase-2-images.md` and execute it. Generate the ChatGPT/Perplexity prompt for the manager to run externally.
4. Read `tiktok_module/procedures/produce-carousel/phase-3-assembly.md` and execute it. Generate the Gemini prompt for the manager to run externally.
5. Read `tiktok_module/procedures/produce-carousel/phase-4-posting.md` and execute it.

Each phase's procedure is self-contained — follow it as written. Always present options and let the manager choose at approval gates.
