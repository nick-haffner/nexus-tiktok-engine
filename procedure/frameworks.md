# Frameworks, Angles & Formats

This is a living catalog. Add new entries as they are used or invented. Claude should reference this file during Phase 0 ideation and propose additions when a new concept is created.

---

## Frameworks

A framework defines the core contrast or structure of the carousel.

| Framework | Description | Example |
|-----------|-------------|---------|
| **Local vs Tourist** | Contrasts tourist traps with local-approved alternatives. High-contrast, opinionated. | Seattle, Dallas, Austin |
| **Worth It** | Positive recommendations only — no traps. Validates places that are popular for good reason. | San Francisco |
| **Overrated vs Underrated** | Similar to Local vs Tourist but broader — doesn't require the location to be a "tourist trap," just overhyped. | (unused — available) |
| **The 24-Hour Test** | "You have 24 hours in [City]." Each slide is a time block (Morning, Afternoon, Evening, Dinner, Late Night) with a "waste it" and a "nail it" choice. Adds urgency and FOMO — the time pressure makes every rec feel higher-stakes. Best paired with split format. | Los Angeles |

## Angles

An angle defines the scope and lens of the carousel.

| Angle | Description | Example |
|-------|-------------|---------|
| **Broad city guide** | Covers multiple categories across the city (food, nightlife, views, etc.). | Seattle, Dallas, Phoenix |
| **Category deep-dive** | All slides focus on one topic in one city (e.g., "Austin BBQ: The Real Guide"). | (unused — available) |
| **Worth It list** | Curated list of recommendations without contrasting pairs. | San Francisco |

## Slide Formats

A format defines how content is laid out across slides.

| Format | Description | Example |
|--------|-------------|---------|
| **Combined** | Tourist and Local on the same slide (split image, top/bottom). | Austin, Phoenix |
| **Split** | Tourist and Local on separate consecutive slides. More visual breathing room. | Seattle |
| **Single-point** | Each slide is one recommendation (no contrast pair). Used with Worth It or list frameworks. | San Francisco |
